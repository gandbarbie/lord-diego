## Why This Project

This project is a fully custom, single-file portfolio built to feel like an interactive security operations interface.

It includes:
- Animated boot sequence and matrix background
- Interactive terminal with commands and autocompletion
- Service catalog with direct WhatsApp message flow
- Background drill soundtrack with mute/unmute control
- Live threat-map and SOC-style event feed simulation
- Responsive layout, reduced-motion support, and keyboard accessibility

## Quick Start

```bash

git clone <your-repo-url>

# 2) Enter folder
cd lord-diego

# 3) Open locally
xdg-open lord-diego-v2.html
```

## Background Audio

- Active track: `assets/audio/hotel-lobby-colors-show-fast.mp3`
- UI control: floating `MUSIC` button (bottom-left)
- Behavior: starts on first interaction, then supports mute/unmute toggle

## Terminal Commands

```bash
whoami, skills, services, exploits, bounties, contact
certs, arsenal, hof, ls, pwd, date, uname, ifconfig
scan <target>, trace <ip>, goto <section>
theme <gold|red|green>, missions
feed <pause|resume|clear|status>
matrix, clear, help
```

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl/Cmd + K` | Open command deck |
| `Tab` | Command autocomplete |
| `Arrow Up / Down` | Terminal history |
| `Esc` | Close deck/modal layers |

## Customization

### Core brand colors

Edit `:root` in [`lord-diego-v2.html`](./lord-diego-v2.html):

```css
:root {
  --gold: #FFD700;
  --red: #FF0033;
  --green: #00FF41;
  --void: #000005;
}
```

### Personal info and channels

Update in HTML:
- Meta tags (`author`, social metadata)

- JSON-LD `sameAs` links
