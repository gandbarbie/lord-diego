## Terminal Commands

```bash
whoami      # About the researcher
skills      # Skill matrix
exploits    # Bug bounty cases
bounties    # Financial summary
contact     # Contact channels
certs       # Certifications
arsenal     # Tools list
hof         # Hall of Fame
scan <target>   # Simulate scan
trace <ip>      # Simulate trace
goto <section>  # Navigate
theme <color>    # Change theme
missions    # Recent ops
feed        # Feed control
matrix      # Visual effect

help        # Command list
```
